# React XD Assignment

## 📌 Project Overview
This project was developed as part of the XD assignment using **React**.  
It demonstrates core React concepts such as component-based architecture, state management, and responsive UI design.  
The application has been successfully deployed on **Netlify**.

---

## 🚀 Features
- Built with **React** and **react-scripts**
- Responsive and lightweight design
- Optimized production build
- Deployed live on Netlify

---

## 🛠️ Tech Stack
- **Frontend:** React, CSS
- **Build Tool:** react-scripts
- **Deployment:** Netlify
